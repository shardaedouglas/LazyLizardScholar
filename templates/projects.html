Unplugged Activities

Learn computer science concepts without a computer! These hands-on activities build foundational understanding through play and movement.

Ages 5-8: Foundation Building

Simple, engaging activities that introduce basic programming concepts through play.

Happy Maps

Help Flurb get to the fruit! Learn to give clear directions and understand the importance of step-by-step instructions.

Unplugged Beginner
Materials Needed:
  • • Grid paper
  • • Colored pencils
  • • Arrow stickers
Getting Loopy Dance

Learn about loops through dance! Discover repeating patterns and how to simplify instructions using repetition.

Unplugged Beginner
Materials Needed:
  • • Open space
  • • Music (optional)
  • • Dance move cards
Graph Paper Programming

Create secret drawings using only symbols! Learn to write programs that a friend can follow to recreate your artwork.

Unplugged Intermediate
Materials Needed:
  • • Graph paper
  • • Pencils
  • • Symbol reference sheet

Ages 9-12: Building Complexity

More challenging unplugged activities that introduce advanced programming concepts.

Debugging Relay

Be a code detective! Find and fix bugs in programs to help them work correctly. Learn systematic debugging techniques.

Unplugged Intermediate
Materials Needed:
  • • Buggy program cards
  • • Correction markers
  • • Timer
Songwriting with Functions

Learn about functions by analyzing songs! Discover how choruses work and create your own song with repeating patterns.

Unplugged Intermediate
Materials Needed:
  • • Song lyrics sheets
  • • Paper and pencils
  • • Musical instruments (optional)
Binary Bracelets

Create your own binary code bracelet! Learn how computers store information using only 1s and 0s.

Unplugged Intermediate
Materials Needed:
  • • Beads (2 colors)
  • • String
  • • Binary alphabet chart
